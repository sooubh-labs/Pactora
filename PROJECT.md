# Pactora

**Status:** MVP Complete | Production V1 in Progress
**Tagline:** Never forget a promise.
**Core:** 100% Offline, Privacy-First Commitment Tracker.

## 📌 Quick Summary
Pactora is a Flutter-based utility app for tracking Promises, Borrowed Items, Money IOUs, and People relationships. It uses a high-performance local database (Isar) to ensure no data ever leaves the user's device.

## 🚀 Key Modules
- **Promises:** Manage tasks and commitments.
- **Borrow:** Track lent/borrowed physical objects.
- **Money:** Log informal debts and payments.
- **People:** Centralized history for all contacts.
- **Insights:** Dashboard, Calendar, and Timeline views.

## 🛠 Tech Snapshot
- **Flutter** (UI)
- **Riverpod** (State)
- **Isar** (Local DB)
- **GoRouter** (Nav)
- **Local Notifications** (Reminders)
