Currency Localization Logic

Store-Driven Pricing

Instead of hardcoding, we use the ProductDetails from the Play Store which automatically provides the localized price string (e.g., ₹160, $1.99, €1.89) based on user's store region.

User Preference Alignment

If user is offline, we fallback to a formatted price using the currencySymbol from userPreferencesProvider.
